## Overview

Nova Forge Model Customization is a generally available AWS feature that enables customers to continued pre-training (CPT) on Amazon's proprietary Nova foundation models on SageMaker HyperPod with/without data-mixing.

While model fine-tuning is commonly available for most models on AWS. CPT on propriatary models was not available anywhere. 
This was first of a kind solution to allow users to perform CPT on propriatary models while proctecting IP data and model weights from customer and also protecting customer's data and techniques from model owner and amazon.

Security was the most important aspect of this product. I worked on core engineering team that designed and built this feature on SageMaker Hyperpod, enabling CPT for IP protected models.

I worked on this project as away team member and got org wide recognistion for my leadership and fast delivery despite being new to HyperPod.
