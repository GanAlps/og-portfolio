## Overview

Nova Forge Model Customization is a generally available AWS feature that enables customers to fine-tune Amazon's proprietary Nova foundation models on SageMaker HyperPod using continued pre-training with data-mixing.

I was part of the core engineering team that designed and built this feature, working across the training orchestration and control plane layers.

## Technologies

Kubernetes, Docker, Java, Go, Python, AWS SageMaker, HyperPod
